# Test Fixtures

JSON test fixtures generated from WordPress. Used by both Rust unit tests
(`patina-core`) and PHP extension tests.

## Format

Each file is a JSON array of test cases:

```json
[
  {
    "input": ["<script>alert(1)</script>"],
    "output": "&lt;script&gt;alert(1)&lt;/script&gt;"
  }
]
```

- `input`: array of positional arguments to the WordPress function
- `output`: the expected return value

## Generation

Fixtures are generated by running the WordPress function against a test corpus
**without** the Patina extension loaded:

```bash
docker compose -f profiling/docker-compose.yml exec php-fpm \
    php /app/php/fixture-generator/generate.php --all
```

## When to Regenerate

- When targeting a new WordPress version that changes function behavior
- When adding a new function to the extension
- When new edge cases are discovered
