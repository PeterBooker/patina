This is a classic (pre-Gutenberg) post — no block delimiters, just raw HTML. It exercises the wpautop path and the render-time kses pass that runs on classic content, neither of which touches parse_blocks because the grammar parser short-circuits to a single core/freeform block when there are no delimiters.

Classic posts are rare in new content but extremely common in archive scans of older sites. Many production WordPress installs still have tens of thousands of pre-2018 posts that have never been touched since the Gutenberg migration, and WP renders them through the same wpautop → kses pipeline on every pageload.

  • No block markup
  • Exercises wpautop on newlines
  • Still hits render-time kses for allowed tags
  • Used by the classic-post scenario in the HTTP bench
